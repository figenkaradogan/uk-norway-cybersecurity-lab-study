# Tool Selection & Justification

Tools were selected based on reliability, transparency,
and suitability for controlled enterprise testing.

## Nmap
Used for host discovery, port scanning, and service version detection.
Chosen for its accuracy and industry acceptance.

## Metasploit Framework
Used to validate exploitability of identified services
in a controlled manner.

Exploit modules were selected conservatively to avoid
system instability.

## OWASP ZAP
Used to assess web application exposure and misconfiguration.
Chosen for its focus on passive and active scanning control.

## Nikto
Used to identify common web server misconfigurations
and outdated components.

## PowerShell
Used to verify host configuration, firewall rules,
and system hardening decisions.
